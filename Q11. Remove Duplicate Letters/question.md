Monotonic Stack + Greedy

Issue Description:
Remove duplicate letters so that every letter appears once and the resulting string is the smallest in lexicographical order.

This problem combines frequency counting, greedy decisions, and stack maintenance.

Acceptance Criteria:

Each character appears once

Lexicographically smallest result

Time Complexity: O(n)

Space Complexity: O(n)
