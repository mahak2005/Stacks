Monotonic Increasing Stack

Issue Description:
Given a numeric string, remove k digits such that the resulting number is the smallest possible.

This problem combines greedy strategy with stack-based digit removal.

Acceptance Criteria:

Produces smallest possible number

Handles leading zeros correctly

Returns "0" if all digits are removed

Time Complexity: O(n)

Space Complexity: O(n)
